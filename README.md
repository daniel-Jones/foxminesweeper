Simple Minesweeper written in C++ using the fox toolkit.

You need FOX http://www.fox-toolkit.org/. On arch you can install the 'fox' package.

Compile:

```
cmake CMakeLists.txt
make
```

binary is in bin/

Right click on a tile to flag it.
